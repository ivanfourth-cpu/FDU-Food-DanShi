# 旦食 FDU Food

复旦人的美食点评与指南系统

## 关于

旦食是一个诞生于复旦校内的独立点评项目。我们不追求商业化的宏大，只在乎每一个档口最真实的烟火气。

在这里，没有虚假的商业滤镜，只有复旦同学之间最真诚的"推荐"与"避雷"。

## 功能

- 🍜 **食堂点评** - 分享各食堂档口的用餐体验
- 🏠 **食客楼** - 校外觅食心得交流
- ⭐ **六星评分** - 独特的评分体系，顶流美食值得六星
- 👍 **真诚推荐** - 点赞/踩功能，帮助发现优质美食

## 技术栈

- **前端**: Vue 3 + 原生 CSS
- **后端**: Node.js + Express
- **数据库**: SQLite

## 开发

```bash
npm install
npm start
```
